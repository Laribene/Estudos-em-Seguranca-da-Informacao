Web filter existe para:
- Objeção de conteudo e segurança
- Examina o conteudo da internet para ver se ele deve ou não ser bloqueado
- Aplicam regras para diferentes usuários.
- Censurar a informação(pesado)
- Podem bloquear adware, spam, virus e spyware
- Adicionados em firewalls, servidores proxy, sandbox, wireless access point

Como funciona um web filter?
- Ele pode consultar um banco de dados de URL que lista sites e dominios conhecidos por hospedar malware(black list) ou sites que são livres dos mesmo
- Outra alternativa é um filtro que procura por palavras chaves(falso positivo)
- Outra mais avançada é usar aprendizado de maquina.

Filtros da web estão integrados nos produtos:
- FortiClient
- FortiGate
- FortiAP(to wireless access points)
