Security in Cloud:
- Responsibility between the cloud provider and the customer utilizing the cloud services.
- Includes the physical and logical components.
- IaaS:It is responsability of a vendor the infrastructure and for securing the infrastructure it provides. Como um customer, quando você instala uma ou mais aplicações de virtualização no vendor's cloud infrastructure, você é responsável for securing the acces, o trafego de rede, e data applications.
- custome'rs data and services are moved to the cloud, the customer is responsible for the security of the data.

Vendors supply some form of security tools, however can pose a few problems:
1) These tools provide only a few basic security functions.
2) Many organizations operate in a hybrid world, this is call multi-cloud environment, and it is a problem because has uncoordinated security solutions.
