Agent System Requirements


Processor Requirements:
  • Minimum 1GHz (32 bit)
  • Recommended 2GHz (64-bit) Intel Pentium or equivalent
  • AMD 64 processor
  • Intel 64 processor

RAM Requirements:
  • Minimum
    • 1GB (32-bit)
    • 2GB (64-bit)
  • Recommended
    • 4GB

Espaço em disco:
  • Minimum
    • 1.5GB
  • Recommended
    • 2GB espaço disponivel sem iES, iVP, iAC and iDLP
    • 3 GB espaço disponivel com iES, iVP, iAC and iDLP
    • 8 GB recommended espaço disponivel
