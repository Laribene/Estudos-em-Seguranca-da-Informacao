Provisioning a new Apex One as a Service Account

- Provision a 30 day trial from Trend Micro's website
- All Provisioning is handled via Customer License Portal(CLP)
- Se o cliente ja tem uma conta CLP, eles podem então adicionar o Apex One to it com uma chave de licença
- Novos clientes podem usar a chave de licença para criar uma nova conta CLP





Downloading and Deploying Agents

- Agents são baixados do Apex One Console
- É recomendado que os clientes completem suas configurações iniciais e repackage the agent antes de baixar the agent package
- Clientes can deploy(implantar) the agent installer using qualquer ferramenta que preferir

Architecture Considerations for SaaS

- Update Agents are still supported in SaaS
- Cliente deve instalar update agents first(if applicable), and then configure Apex Onde to use them before deploying the majority of agents
(Update Agents can help reduce WAN bandwidth usage in locations with multiple agents )
